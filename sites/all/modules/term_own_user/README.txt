- Khi chỉnh sửa, cũng hông chi hiển thị các user con
- Khi vào trang User profile thì chỉ hiển thị các term mà user đó tạo ra.
- Khi bỏ chọn vào Term mà user đó đăng ký làm ngành nghề kinh doanh thì ..... đặt các term đó sang 1 tình trạng TREO, nếu có các
 + Cần phát triển thêm khi user đó chọn sang ngành nghề khác thì thông báo cho user lựa chọn 1 trong 2 option: 
	* chuyển tất cả các term cũ sang term root mới hay không
	* Đặt các term củ ở trình trạng treo.
- Khi thêm/ sửa một term mới thì user đó chỉ thấy own term root và các term con mà chính user đó tạo ra.
